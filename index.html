📘 Panduan Google Sheets Setup
Langkah Setup Google Sheets:
1. Buat Google Sheets baru (kosong)
2. Buka Extensions → Apps Script
3. Paste kode Apps Script (tekan tombol di bawah)
4. Klik Run → initSheets untuk membuat 5 sheet otomatis
5. Klik Deploy → New Deployment → Web App
6. Set: Execute as Me • Who has access: Anyone
7. ✅ URL sudah tertanam permanen — aplikasi langsung aktif!
⚠️ Penting: Setiap kali edit Apps Script, harus Deploy ulang (buat deployment baru, bukan update).
